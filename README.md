# Funktions2

A simple Hello World web application.

## Files

- `index.html` - Main HTML file
- `app.js` - JavaScript file with console log
- `app.css` - CSS stylesheet

## Usage

Simply open `index.html` in a web browser to run the application.

## Git Commands

### Create .gitignore (to ignore history and other files)
Create a `.gitignore` file with:
```
.history/
.env
node_modules/
dist/
build/
```

### Initialize Git Repository
```bash
git init
```

### Add Files
```bash
git add .
```

### Commit Changes
```bash
git commit -m "Initial commit"
```

### Rename Branch to Main
```bash
git branch -M main
```

### Add Remote Repository
```bash
git remote add origin https://github.com/Ajaxy12/funktions2.git
```

### Push to GitHub
```bash
git push -u origin main
```

Note: Make sure the repository exists on GitHub before pushing.
